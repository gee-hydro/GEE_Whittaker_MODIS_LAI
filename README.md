# GEE Whittaker Smoothed LAI (2000-2024)

## Usage

GEE调用代码见：[LAI_WHITV016.js](./LAI_WHITV016.js)

## Highlights

- 保护双生长季信号

- 保护干旱响应信号

- EBF质量提升
